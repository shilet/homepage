Attacked Pieces

Challenge to learn how the knight moves fast from one square to another. First you specify the number of pawn in home->settings->blindchess level. Then you should find the minimum number of moves to catch all pawns. This exercise is suitable for beginners when the number of pawns is small (two, or three). The more pawns you select, the more difficult this exercise is.
