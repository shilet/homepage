RealChess

You are given a position and you should find the best move. This move can be either a defending or a winning move. Some moves are very subtle and difficult to find, making this challenge useful for very experienced chess players as well. For players who want more challenge, try blindchess

Example 1

The last move black played is Ke8. Find the best move for white.
Solution is Qxc5. White has to take the bisshop on c5 to get an equal position. This is an example of a rather simple exercise.

Example 2

The last move white played is g4. Find the best move for black.
Solution is b5. This will give black an equal position. Other moves give white a better position, so this is the best move.

Example 3

The last move white played is Kg2. Find the best move for black.
Solution is Rxe1. This move will win a rook.

Example 4

Sometimes more moves should be done. This is an example where three move should be found. Finally white has perpetual and can make a draw.
The last move black played is Qa1+. Find the best move for white.
Best move is Kh2. Black will play b2 now.
Find best move again. Best move now is Qxe6+. Black will play Kh7.
Best move now is Kf5, which will give perpetual check.
